# Stopwatch Web App

A responsive stopwatch web application built with plain HTML, CSS, and JavaScript.

## Features

- Start, pause, reset, and lap controls
- Time display in `HH:MM:SS:MS` format
- Modern dark user interface
- Responsive layout for desktop and mobile
- Static files only, ready for GitHub Pages deployment

## How to Run Locally

1. Download or clone this repository.
2. Open `index.html` directly in your browser.

You can also serve the folder with any simple static server if preferred.

## GitHub Pages Deployment

1. Push the project to a GitHub repository.
2. Open the repository on GitHub.
3. Go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the branch containing your files, typically `main`.
6. Set the folder to `/ (root)` and save.
7. Wait for GitHub Pages to publish the site, then open the generated URL.
